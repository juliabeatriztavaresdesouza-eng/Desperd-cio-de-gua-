# Desperdício-de-água- 
o combate ao desperdício de agua.

## oque seria o projeto?

o projeto está ligado a atentar as pessoas que não entendem a importância da economia de água, é de fato muito importante saber oque se deve fazer a respeito, e é isso que estamos a apresentar a seguir.

## oque está sendo mostrado?

no projeto, está sendo mostrando soluçoes dos problemas  que estão sendo enfrentados, assim sendo a forma de enfrentalos para acabar com o desperdício de água.


* O mapa da nossa ideia (projeto canvas).
* o roteiro da apresentação.
* os problemas e soluções.
* e idéias.
